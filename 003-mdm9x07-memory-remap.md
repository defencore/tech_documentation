# MDM9x07 MEMORY REMAP

For memory remap we need Qualcomm MIBIB partitioning file for your device:

- partition_complete_p4K_b256K.mbn
- partition_complete_p2K_b128K.mbn
- or dump it using edl r mibib mibib.bin
